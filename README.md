# affiliate_audit
Audits for company and personal family relationship. 

## Running

```
cd affiliate_audit
```

```
docker build -t IMAGE_NAME .
```

```
docker build -t aff_img .
```

```
docker run -d -it -p HOST_PORT:CONTAINER_PORT --name CONTAINER_NAME IMAGE_NAME
```

```
docker run -d -it -p 8005:8000 --name aff_inst add_img
```
